# 🚀 AI & Machine Learning Portfolio

Welcome to my central repository for Artificial Intelligence, Machine Learning, and TinyML projects. This workspace contains a collection of end-to-end solutions, ranging from financial time-series forecasting to hardware-level edge AI.

## 📁 Repository Projects

Below is an overview of the projects included in this repository. Click on any project name to explore its code, architecture, and specific documentation.

### 🎙️ [TinyML-Voice-command](./TinyML-Voice-command)
*   **Domain:** Embedded AI / Edge Computing
*   **Tech Stack:** MicroPython, Scikit-Learn (LDA, Logistic Regression), Optuna
*   **Description:** A real-time Keyword Spotting (KWS) system aggressively optimized for microcontrollers. Features a custom audio processing pipeline that compresses temporal features into a highly efficient `<3KB` inference model.

### 🕵️‍♂️ [Fraud Detection](./Fraud%20Detection)
*   **Domain:** Machine Learning / Tabular Data
*   **Tech Stack:** Python, XGBoost, Optuna
*   **Description:** An advanced classification pipeline designed to detect fraudulent activities. Utilizes rigorous hyperparameter optimization (via Optuna) to maximize recall and handle highly imbalanced datasets effectively.

### 📈 [Bitcoin Time-Series Forecasting A Machine Lear...](./Bitcoin%20Time-Series%20Forecasting%20A%20Machine%20Lear...)
*   **Domain:** Financial Machine Learning / Predictive Modeling
*   **Tech Stack:** Python, Time-Series Analysis
*   **Description:** A predictive modeling project focused on cryptocurrency historical datasets. Designed to analyze trends and forecast Bitcoin time-series data using robust machine learning algorithms.

### 🍎 [Fruit Quality Inspection](./Fruit%20Quality%20Inspection)
*   **Domain:** Computer Vision / Quality Control
*   **Tech Stack:** Python, Machine Learning
*   **Description:** An automated inspection system designed to classify and evaluate the quality of fruits, streamlining agricultural and quality assurance processes.

### 🎨 [RGB Color Detection(Micro)](./RGB%20Color%20Detection(Micro))
*   **Domain:** Embedded Systems / Microcontrollers
*   **Tech Stack:** MicroPython, Sensors
*   **Description:** A hardware-interfacing project focused on reading and processing RGB color values directly on a microcontroller, bridging the gap between raw sensor data and algorithmic logic.
